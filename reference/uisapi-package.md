# uisapi: Access the UNESCO Institute for Statistics API

Retrieve data from the UNESCO Institute for Statistics (UIS) API
<https://api.uis.unesco.org/api/public/documentation/>. UIS provides
public access to more than 4,000 indicators focusing on education,
science and technology, culture, and communication.
